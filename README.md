# ToDo — Твой список задач

Простое, красивое и минималистичное приложение для ведения дел.  
Тёмная тема, акцентный голубой цвет, шрифт Russo One и приятные анимации.

![Скриншот приложения](./img/image.png)

## Что уже работает

- Добавление новых задач
- Удаление задач
- Редактирование задач
- Отметка задачи как выполненной (чекбокс + зачёркивание)
- Сообщение о статусе («Задач нет. Отдыхай!», «Работаем!» и др.)
- Кастомный чекбокс в виде круга с анимацией

## Что находится в разработке / планируется

- [ ] Особый дизайн выполненных задач (**в разработке**)
- [ ] Счётчик задач («Осталось 3 задачи», «Всё сделано!»)
- [ ] Сохранение задач в localStorage (чтобы не пропадали при перезагрузке)
- [ ] Кнопка «Очистить выполненные»
- [ ] Анимации появления/удаления задач
- [ ] Поддержка тёмной/светлой темы (переключатель)
- [ ] Возможно: категории / приоритеты / дедлайны

## Технологии

- HTML5 + CSS3 (flex, кастомные чекбоксы, псевдоэлементы)
- Vanilla JavaScript (без фреймворков)
- Шрифт: [Russo One](https://fonts.google.com/specimen/Russo+One)
- Сброс стилей через reset.css
